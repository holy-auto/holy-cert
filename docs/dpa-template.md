# Data Processing Agreement (DPA) — テンプレート

Ledra (受託者) と お客様 (委託元) との間の個人データ取扱いに関する
データ処理契約書テンプレート。各加盟店との契約締結用。

> **本文書はテンプレート (法務未承認) です。実契約に使用する前に、
> 必ず法務レビューを受けてください。**

---

## 第 1 条 (定義)

本契約において、以下の用語は次の意味を有する:

- **「個人データ」**: 個人情報保護法 (令和 4 年改正) 第 2 条第 7 項に
  定める個人情報データベース等を構成する個人情報。
- **「処理」**: 個人データに対する一切の作業行為 (収集、保存、検索、
  更新、開示、消去等) をいう。
- **「サブプロセッサ」**: Ledra が個人データの処理を再委託する第三者。
- **「データ主体」**: 個人データにより識別される自然人 (お客様の顧客)。

## 第 2 条 (役割)

- **委託元 (お客様)** は本サービスを利用して取り扱う個人データの
  「個人情報取扱事業者」(Controller) である。
- **受託者 (Ledra)** は委託元の指示に基づき個人データを処理する
  「処理者」(Processor) である。

## 第 3 条 (処理の対象 / 範囲 / 目的)

| 項目 | 内容 |
|------|------|
| 対象データ主体 | お客様の顧客 (車両所有者、施工依頼者) |
| 処理対象 | 氏名、連絡先 (電話末尾 4 桁ハッシュ + メール)、施工履歴、車両情報、決済情報 (Stripe 経由) |
| 処理目的 | 施工証明書発行、顧客ポータル提供、課金処理、サービス改善のための統計分析 |
| 処理期間 | 委託元のサービス利用期間中 + 解約後 30 日 (バックアップ含む) |

## 第 4 条 (受託者の義務)

Ledra は以下を遵守する:

1. 委託元の文書化された指示に従ってのみ個人データを処理する
2. 個人データへのアクセスを業務上必要な従業員に限定する
3. 適切な技術的・組織的安全管理措置を講じる
   (詳細: `docs/security-overview.md`)
4. 漏洩等の事故発生時は **24 時間以内** に委託元に通知する
5. 委託元の請求に応じてデータ主体の権利行使 (開示・訂正・削除・
   利用停止) に協力する
6. 委託元の請求に応じて監査を受ける (年 1 回まで、合理的な範囲)

## 第 5 条 (サブプロセッサ)

Ledra は以下のサブプロセッサを使用する:

| サブプロセッサ | 役割 | 所在地 |
|---------------|------|--------|
| Supabase Inc. | DB / 認証 | アメリカ合衆国 (東京リージョン) |
| Vercel Inc. | アプリホスティング | アメリカ合衆国 (東京リージョン) |
| Stripe Payments Japan K.K. | 決済処理 | 日本 |
| Resend Inc. | メール配信 | アメリカ合衆国 |
| Cloudflare Inc. | CDN / 動画ストリーミング | グローバル |
| Upstash Inc. | レート制限 / 非同期ジョブ | アメリカ合衆国 |
| Functional Software Inc. (Sentry) | エラー監視 (PII マスク済) | アメリカ合衆国 |

サブプロセッサの追加・変更時は **30 日前に** 委託元に通知する。
委託元は合理的な反対理由がある場合、書面で異議を申し立てることができる。

## 第 6 条 (越境移転)

サブプロセッサの一部はアメリカ合衆国に所在する。Ledra は以下により
個人情報保護法第 28 条 (外国にある第三者への提供) を遵守する:

- 各サブプロセッサと **個人情報保護法令に準拠した契約 (DPA)** を締結
- 越境移転先の **国名と保護水準** をデータ主体に通知 (利用規約)
- 必要に応じて Standard Contractual Clauses (SCC) または同等の措置

## 第 7 条 (データ主体の権利行使)

委託元から開示・訂正・削除等の請求を受領した場合、Ledra は次の
仕組みでこれに対応する:

| 請求種別 | 対応エンドポイント | 完了目安 |
|---------|------------------|---------|
| 開示 (アクセス権) | `/api/customer/data-export` | 即時 |
| 訂正 | お客様管理画面 | 即時 |
| 削除 / 利用停止 | `/api/customer/data-deletion` | 30 日以内 |
| 利用履歴開示 | `/api/customer/audit-log` | 即時 |

## 第 8 条 (契約終了時の処理)

本契約の終了時、Ledra は委託元の指示に従い:

1. 個人データを **30 日以内** に削除または委託元に返還
2. バックアップに残存するデータは保持期間 (90 日) 満了後に削除
3. 削除完了報告書を委託元に提出

## 第 9 条 (準拠法・管轄)

本契約は **日本法** を準拠法とし、紛争は **東京地方裁判所** を
第一審の専属的合意管轄裁判所とする。
