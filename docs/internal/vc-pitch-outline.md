各ページに「目的」「載せる要素」「想定セリフ (口頭補足)」を併記しています。
社内デザイナーがそのままスライド化できる粒度を意識しています。

## 1. 表紙

- **目的**: ブランド第一印象 + ワンフレーズで事業を伝える
- **要素**:
  - Ledra ロゴ (ダークテーマ #060a12 / アクセント #0071e3)
  - Tagline: 「車の施工を、デジタルで証明する。」
  - サブコピー: "施工店 SaaS × 施工証明書ネットワーク"
  - 提出先 / 日付 / 会社名 (株式会社HOLY)

---

## 2. チーム

- **目的**: "なぜ我々がやるのか" を顔と一次情報で語る
- **要素**:
  - 堀越 友輔 (CEO) — 顔写真 + 自動車施工現場 (detailing_holy) の写真
  - <PM / Engineer / Sales> — 役割と一行プロフィール
  - キーメッセージ: "施工店オーナー出身の Founder が現場の痛みを起点に開発"

---

## 3. 業界課題① 施工店の業務分散

- **目的**: 課題の深さを「数字」で見せる
- **要素**:
  - 現場写真 (紙の施工記録 / LINE スクショ / Excel 台帳) を 3 枚並べる
  - 数字: 1 件あたりの書類対応時間 / 月間の工数ロス / 二重入力の発生回数
  - キーフレーズ: "真面目に施工する事業者ほど書類で消耗する"

---

## 4. 業界課題② 保険・中古車流通の検証不能

- **目的**: 流通レイヤの空白を提示
- **要素**:
  - フロー図: 施工店 → 保険会社 / 中古車市場 の情報非対称
  - 数字: 不正コーティング請求の推定額 / 中古車査定ブレ率
  - キーフレーズ: "施工した事実が後から検証できない"

---

## 5. Why Now

- **目的**: 今やる必然性を示す
- **要素**:
  - SDV (Software Defined Vehicle) 化と整備履歴デジタル化の潮流
  - 車検制度 / OBD 検査の電子化スケジュール
  - 損保業界の不正請求対策強化
  - 中古車流通の透明性ニーズ (UCDA 等)
  - キーフレーズ: "車の履歴がデジタル化される 10 年に 1 度のタイミング"

---

## 6. プロダクト概観

- **目的**: "何を作っているか" を 1 枚で
- **要素**:
  - 4 ポータル図 (Admin / Agent / Insurer / Customer) + モバイル
  - `/admin/jobs/[id]` 案件ワークスペースのスクショ
  - 主要数字: 400+ API / 130+ migrations / 2,000+ tests
  - キーフレーズ: "業務 SaaS と証明書ネットワークが地続きの 1 プロダクト"

---

## 7. 施工証明書の差別化

- **目的**: 模倣困難性を技術で示す
- **要素**:
  - QR / 公開 URL `/c/[public_id]` のスクショ
  - Polygon アンカリングのフロー図 (写真 SHA-256 → on-chain)
  - C2PA / NFC タグ連携
  - キーフレーズ: "発行して終わり、ではなく検証され続ける証明書"

---

## 8. デモ動画 / スクリーンショット

- **目的**: 触感を見せる
- **要素**:
  - 5 分動画 (`/video`) のダイジェスト 30 秒
  - スマホで証明書を見るシーン
  - 保険会社が証明書を検索するシーン
  - キーフレーズ: "現場で動くプロダクト、業界横断で参照される証明書"

---

## 9. ビジネスモデル

- **目的**: ARPU と拡張余地を示す
- **要素**:
  - プラン別 ARPU: フリー ¥0 / スターター ¥9,800 / スタンダード ¥24,800 / プロ ¥49,800
  - アドオン: ブランド証明書 (¥3,300〜) / 追加店舗 (¥4,980/店舗) / 導入伴走
  - 中長期: 保険会社課金 / 中古車流通課金 / トランザクション手数料
  - キーフレーズ: "サブスク + テンプレート + 流通手数料の 3 階建て"

---

## 10. 市場規模 (TAM/SAM/SOM)

- **目的**: 大きさを示す
- **要素**:
  - TAM: 国内自動車整備市場 5.5 兆円 (経産省 / 日整連)
  - SAM: コーティング・PPF・板金 の DX 化対象 X 兆円
  - SOM: 5 年で <Y> 万店舗 × ARPU = <ARR> を狙う
  - キーフレーズ: "整備 5.5 兆円市場の "履歴" レイヤを取りに行く"

---

## 11. GTM 戦略

- **目的**: どう広げるかを 3 段で示す
- **要素**:
  1. 直販: detailing_holy のネットワーク + 業界イベント
  2. 代理店: Agent ポータルでパートナーが施工店を紹介
  3. 流通プル: 保険会社・中古車事業者 → 加盟施工店から逆引きで Ledra 採用
  - キーフレーズ: "施工店だけを売り込むのではなく、業界の上流から圧力を作る"

---

## 12. トラクション

- **目的**: 進捗の証明
- **要素**:
  - 店舗数推移グラフ
  - MRR / 月間証明書発行数の推移
  - 主要パイロット (保険会社 X 社 / OEM Y 社)
  - 顧客の声 (`docs/marketing/case-study-guide.md` から 1-2 件)

## 13. 競合マップ

- **目的**: 立ち位置を可視化
- **要素**:
  - 縦軸: 施工店業務 SaaS ↔ 業界横断ネットワーク
  - 横軸: 自前 PDF ↔ デジタル証明書 + 検証
  - 既存プレイヤー (車整備 SaaS / コーティング DIY 系 / 紙の施工証明書) を散布
  - Ledra は右上に単独でポジション
  - キーフレーズ: "業務 + 証明書 + 流通の 3 つを統合する事業者は存在しない"

---

## 14. ロードマップ

- **目的**: 中期計画の透明性
- **要素**:

  - キーフレーズ: "プロダクトはすでにある。次の 24 ヶ月は流通ネットワーク拡張"
