## 1. ブランド証明書オプション利用規約

### 第1条（定義）
本規約において、以下の用語は以下の意味で使用します。

1. 「ブランド証明書オプション」とは、CARTRUST が提供する施工証明書テンプレートのカスタマイズサービスを指します。
2. 「ライトプラン」とは、既製テンプレートを利用したカスタマイズサービスを指します。
3. 「プレミアムプラン」とは、オリジナルテンプレートの制作代行サービスを指します。
4. 「加盟店」とは、本オプションを契約するテナント（法人または個人事業主）を指します。
5. 「テンプレート」とは、施工証明書の見た目・レイアウト・文言等を定義するデータを指します。

### 第2条（サービス内容）
1. ライトプランでは、CARTRUST が提供する既製テンプレートから選択し、管理画面を通じてロゴ・配色・文言等のカスタマイズを行えます。
2. プレミアムプランでは、CARTRUST の制作担当が加盟店のヒアリング内容に基づき、オリジナルテンプレートを制作します。
3. いずれのプランにおいても、CARTRUST の施工証明書構造（必須表示項目を含む）を維持した範囲でのカスタマイズとなります。

### 第3条（料金および支払い）
1. 料金は、初期費用（一括払い）および月額費用（毎月払い）で構成されます。
2. 表示価格はすべて税込です。
3. 支払いは、クレジットカード決済（Stripe）によるものとします。
4. 月額費用は、サービス開始日を起算日として毎月自動的に請求されます。
5. プレミアムプランの月額費用は、テンプレートの公開日から課金を開始します。

### 第4条（提供範囲の制限）
1. 以下の項目は、施工証明書としての信頼性を担保するため、非表示・削除することはできません。
   - お客様名（証明書宛名）
   - 車両情報
   - 施工内容
   - 証明書番号
   - CARTRUST認証マーク
2. テンプレートの変更は、CARTRUST が定める構造化カスタムの範囲内に限ります。
3. テンプレートのソースコード・データの提供は行いません。

### 第5条（知的財産権）
1. 加盟店が提出するロゴ・画像等の素材に関する著作権・商標権等の権利は、加盟店に帰属します。
2. 加盟店は、提出する素材が第三者の権利を侵害しないことを保証します。
3. CARTRUST が制作したテンプレートのデザイン・レイアウト・構造に関する著作権は CARTRUST に帰属します。
4. 加盟店は、契約期間中に限り、テンプレートを施工証明書の発行目的で使用する非独占的な利用権を有します。

### 第6条（法務文言に関する免責）
1. 加盟店が指定する保証文言・注意文言・免責文言等の内容の法的妥当性については、加盟店の責任において確認するものとします。
2. CARTRUST は、加盟店が指定した文言の法的有効性・適法性について一切の保証を行いません。
3. 本オプションには弁護士レビュー・法務チェックは含まれません。
4. 法務文言に起因するトラブルについて、CARTRUST は一切の責任を負いません。

### 第7条（修正対応）
1. ライトプランでは、管理画面を通じて加盟店自身がいつでも設定変更を行えます。
2. プレミアムプランでは、初期費用に初回修正1回分を含みます。
3. プレミアムプランの公開後修正は、公開日から2週間以内に限り、軽微な修正1回を含みます。
4. 上記を超える修正は、追加作業費が発生します。
5. 追加作業費は、CARTRUST が作業内容に応じて見積もり、加盟店の承認を得た上で請求します。

### 第8条（解約）
1. 加盟店は、Stripe の課金ポータルを通じていつでも月額費用の解約手続きを行えます。
2. 解約は、現在の課金期間の終了日をもって有効となります（日割り返金は行いません）。
3. 解約後は、CARTRUST 標準テンプレートに自動的に切り替わります。
4. カスタムテンプレートのデータは、解約後30日間保持します。30日経過後はアーカイブされ、復元には再契約が必要です。
5. 初期費用の返金は行いません。

### 第9条（キャンペーン）
1. CARTRUST が実施するキャンペーンの条件は、各キャンペーンの告知に従います。
2. キャンペーン適用の場合、最低利用期間（6ヶ月）が設定される場合があります。
3. 最低利用期間内の解約の場合、キャンペーン割引分の全額を請求する場合があります。

### 第10条（支払い遅延）
1. 月額費用の支払いが遅延した場合、Stripe による自動リトライ（最大3回）を行います。
2. すべてのリトライが失敗した場合、7日間の猶予期間を設けます。
3. 猶予期間経過後もお支払いが確認できない場合、テンプレートの利用を一時停止（CARTRUST標準テンプレートに切替）します。
4. 一時停止後30日以内にお支払いが確認できた場合、カスタムテンプレートを復元します。

```

    追加の修正は別途費用が発生します。
```
