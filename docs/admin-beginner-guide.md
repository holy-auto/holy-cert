# はじめての方向け Ledra 操作マニュアル

> **このマニュアルは、パソコンや IT 用語に詳しくない方でも、Ledra（レドラ）を毎日の業務でひととおり使えるようにする手引きです。**
>
> 専門用語はできるだけ使わず、画面で実際に表示される文字（ボタン名・項目名）と同じ言葉でご案内しています。
>
> **対象**: 整備工場・コーティング店・ボディリペア店の現場スタッフ／オーナー
> **読み方**: 上から順に読まなくても大丈夫です。困ったときに「目次」から該当ページを開いてください。

## 1. Ledra（レドラ）とは何ですか？

Ledra は、**お店のパソコンや iPad で使える「お店の台帳」のようなしくみ** です。今まで紙やノートに書いていた

- お客さまの名前と連絡先
- お車の車種・ナンバー
- 「いつ・どんな施工をしたか」の証明
- 予約・請求書

をすべてひとつの画面でまとめて管理できます。

**特に便利なところ**

| 今までの困りごと | Ledra で変わること |
|---|---|
| 紙の証明書がよごれる・なくなる | 画面で何度でも見られて、PDF（ピーディーエフ）として印刷もできる |
| 「いつ施工したっけ？」が思い出せない | 過去の施工がぜんぶ画面に残る |
| お客さまから「証明書をもう一度ほしい」と言われる | お客さまのスマホで自分で見られる仕組みがある |
| 請求書の作成に時間がかかる | 施工内容からそのまま請求書がつくれる |

## 2. 準備するもの

| 必要なもの | 説明 |
|---|---|
| **パソコン または iPad** | カメラ付きの iPad が一番おすすめです。スマホでも使えます。 |
| **インターネットの線（Wi-Fi など）** | つながっていないと使えません。 |
| **ID（メールアドレス）と パスワード** | お店の代表者からもらってください。初めて使う方には、代表者から「招待メール」が届きます。 |
| **ブラウザ（Chrome／Safari／Edge）** | 普段インターネットを見るときに使っているソフトです。新しいバージョンを使ってください。 |

> **メモ**: パスワードは紙のメモではなく、頭の中で覚えるか、お店で決めた安全な場所に保管してください。他人に教えないでください。

## 3. ログイン（初めて画面を開く）

### 3.1 画面を開く

1. パソコンで **Chrome（クローム）** や **Safari（サファリ）** を開きます。
2. 画面いちばん上の住所を入れる欄に、**お店専用のアドレス**（例：`https://app.ledra.jp/login` のようなもの）を入れます。アドレスはお店の代表者からお聞きください。
3. **「ログイン」** という画面が出ます。

### 3.2 メールアドレスとパスワードを入れる

| 入れる場所 | 入れるもの | 例 |
|---|---|---|
| **Email** | あなたのメールアドレス | `tanaka@your-shop.co.jp` |
| **Password** | あなたのパスワード | （お店で決めたもの） |

入れ終わったら、いちばん下の **［ログイン］** ボタンを押します。

### 3.3 うまく入れないとき

- **「メールアドレスかパスワードが違います」と出る** → 大文字・小文字をまちがえていないか確認してください。`A` と `a` はちがう文字です。
- **そもそも招待メールが来ていない** → お店の代表者に「Ledra に追加してください」とお伝えください。
- **パスワードを忘れた** → ログイン画面の「パスワードを忘れた」を押すと、メールでリセット手順が届きます。

## 4. 画面の見方

ログインに成功すると、まず **ダッシュボード** という画面が出ます。「お店の今日の様子がひとめでわかるホーム画面」と思ってください。

```
┌──────────────────────────────────────────────────────┐
│ ［上の帯］お店の名前 ／ あなたの名前 ／ 設定アイコン  │
├────────┬─────────────────────────────────────────────┤
│ ［左メニュー］│  ［中央エリア］                        │
│              │                                        │
│ ・ダッシュボード │  今日の予定・売上・タスク           │
│ ・予約管理    │                                        │
│ ・証明書      │                                        │
│ ・車両管理    │                                        │
│ ・顧客管理    │                                        │
│ ・請求・帳票  │                                        │
│ ・設定        │                                        │
│ ……          │                                        │
└────────┴─────────────────────────────────────────────┘
```

### 4.1 左メニュー（よく使うもの）

| メニュー | こんなときに押す |
|---|---|
| **ダッシュボード** | お店全体の状況を見たい |
| **予約管理** | 今日来るお客さまを確認したい／予約を入れたい |
| **証明書** | 施工後の証明書をつくりたい／お渡ししたい |
| **車両管理** | お車を新しく登録したい／前回の整備内容を見たい |
| **顧客管理** | お客さまを新しく登録したい／連絡先を見たい |
| **請求・帳票** | 請求書をつくりたい・印刷したい |
| **設定** | お店の住所、ロゴ、メンバーを変えたい |

> **コツ**: 迷ったらまず **ダッシュボード** に戻ると安心です。左メニューの一番上にあります。

## 5. 1 日の仕事の流れ（おすすめの順番）

朝から夜まで、こんな順番でお使いいただくとスムーズです。

```
朝   ① ログインする
     ② ダッシュボードで「今日の予定」を確認
来店  ③ お客さまの情報・お車を確認（または新規登録）
作業中 ④ 写真を撮る（あとで証明書につけます）
作業後 ⑤ 証明書をつくる（写真をつけて発行）
お渡し ⑥ 証明書をお客さまにお渡し（メール／QR コード／印刷）
会計  ⑦ 請求書をつくる・お会計
夕方  ⑧ 明日の予約を確認 → ログアウト
```

それぞれの詳しいやりかたは、このあとの章で説明します。

## 6. お客さまを登録する

新しいお客さまが来店されたら、最初に登録します。

### 6.1 手順

1. 左メニューの **［顧客管理］** を押す。
2. 画面の右上にある **［新規顧客を登録］** ボタンを押す。
3. 出てきたフォームに、お客さまの情報を入れる（下の表を参考に）。
4. いちばん下の **［保存する］** ボタンを押す。

### 6.2 入れる項目

| 項目 | 入れ方の例 | 必須？ |
|---|---|---|
| **氏名** | `山田 太郎`（姓と名のあいだに半角スペース） | はい |
| **フリガナ** | `ヤマダ タロウ`（カタカナで） | おすすめ |
| **メールアドレス** | `taro@example.com`（@ がある形） | できれば |
| **電話番号** | `090-1234-5678`（ハイフンあり） | はい |
| **郵便番号** | `123-4567` | できれば |
| **住所** | `東京都渋谷区...` | できれば |
| **備考** | `アレルギーあり` `紹介者: 鈴木さま` など、お店で覚えておきたいこと | 任意 |

> **メモ**: メールアドレスは「お客さまにスマホで証明書をお渡しする」ときに使います。空欄でも登録できますが、入れていただくと便利です。

## 7. お車を登録する

お客さまの登録ができたら、続けて車を登録します。

### 7.1 手順（カンタンな方法）

1. 左メニューの **［車両管理］** を押す。
2. 右上の **［車両を登録］** を押す。
3. 「**車検証から自動入力**」のところで、車検証（しゃけんしょう）の写真を撮るか QR を読みこむと、自動で項目が埋まります。
4. 自動で入らなかったところは手で入れます。
5. **［保存する］** を押す。

### 7.2 手で入れる場合の項目

| 項目 | 例 |
|---|---|
| **メーカー** | `Toyota` |
| **車種** | `Toyota Prius` |
| **ナンバー** | `水戸 300 あ 12-34`（プレートに書いてあるとおり） |
| **年式** | `2022` |
| **車台番号（VIN）** | 例 `JF1GP7LD7EG000001`（17 文字、車検証に記載） |
| **車両サイズ** | 「軽」「普通車」など |
| **持ち主（顧客）** | 検索欄に名前を入れて選ぶ |

> **コツ**: ナンバーは「ひらがな」もそのまま入れて大丈夫です。

---

## 8. 予約を入れる

### 8.1 手順

1. 左メニューの **［予約管理］** を押す。
2. **カレンダー** が表示されるので、予約したい日付・時間の枠を押す。
3. 予約フォームが出てくるので、下のとおりに入れる。
4. **［保存する］** を押す。

### 8.2 入れる項目

| 項目 | 説明 |
|---|---|
| **開始** | 予約の開始時刻（例：`10:00`） |
| **終了** | 予約の終了時刻（例：`12:00`） |
| **顧客** | 検索欄で名前を入れて選ぶ。新規のお客さまは先に章 6 で登録 |
| **車両** | 同じく、検索して選ぶ |
| **メニュー** | 例：`ガラスコーティング`、`ボディ磨き` |
| **見積金額** | 税込の合計金額 |
| **備考** | 「左フェンダーにキズあり」など覚え書き |

予約後は、ダッシュボードや予約管理画面の **カレンダー** で当日の予定として表示されます。

### 8.3 予約をキャンセルしたいとき

予約を選び、**［キャンセル］** を押します。理由を入れる欄が出るので、わかる範囲で書いて保存してください（あとで振り返りに役立ちます）。

## 9. 証明書をつくる（写真つき）

施工が終わったら、お客さま向けの **施工証明書** をつくります。これが Ledra のいちばん大事な機能です。

### 9.1 手順

1. 左メニューの **［証明書］** を押す。
2. 右上の **［新規発行］** を押す。
3. お車を選ぶ（検索欄に「車のナンバー」または「お客さま名」を入れて選ぶ）。
4. 施工内容を入れる（章 9.2 参照）。
5. 写真をアップロードする（章 9.3 参照）。
6. いちばん下の **［発行］** を押す。

### 9.2 入れる項目（施工内容）

| 項目 | 説明・例 |
|---|---|
| **施工内容（自由記述）** | やったことを文章で書く。例：「下地処理 → ガラスコーティング 2 層 → 仕上げ拭き上げ」 |
| **実施日** | 作業した日 |
| **有効条件（テキスト）** | 例：`半年ごとにメンテ推奨` |
| **有効期限** | 「いつまで効きますか？」のおおよその目安 |
| **保証期間（終了日）** | 保証が切れる日 |
| **保証対象外の条件・注意事項** | 例：`飛び石による損傷、経年劣化、不適切な洗車方法による損傷等` |
| **備考・特記事項** | その他、お客さまに伝えたいこと |

> **コツ**: 文章は完璧でなくても大丈夫です。「だいたい合っていればよし」でかまいません。あとから直せます。

### 9.3 写真をのせる

「**写真をアップロード**」のところに、施工中・施工後の写真をドラッグ＆ドロップ（マウスでつかんで離す）するか、押して選びます。

- iPad の場合は、その場で撮った写真をそのまま選べます。
- 1 枚あたり **10 MB** くらいまでが目安です。
- 何枚でも追加できますが、**3〜5 枚** くらいがおすすめです（多すぎるとお客さまが見にくい）。

### 9.4 発行したあと

発行が完了すると、**証明書番号** が出ます（例：`LEDRA-DEMO-0001`）。
この番号があれば、あとから何度でも同じ証明書を呼び出せます。

## 10. お客さまに証明書をお渡しする

発行した証明書は、3 つの方法でお渡しできます。

### 10.1 メールで送る（おすすめ）

1. 証明書を開く。
2. **［メール送信］** を押す。
3. お客さまのメールアドレスが入っていれば、自動で送られます。

### 10.2 QR コードを見せる

1. 証明書画面の右上にある **QR コード** をお客さまのスマホで読みとってもらう。
2. お客さまのスマホに証明書ページがそのまま表示されます。

> **メモ**: あとから紛失されても、同じ QR から何度でも見られます。

### 10.3 紙で印刷する

1. 証明書画面の **［PDF 出力］** を押す。
2. PDF が画面に出るので、ふつうのプリンターで印刷できます。
3. A4 サイズが標準です。

## 11. 請求書（せいきゅうしょ）を出す

### 11.1 手順

1. 左メニューの **［請求・帳票］** を押す。
2. 右上の **［新規請求書］** を押す。
3. お客さま・お車・金額を入れる。証明書から自動で読みこむこともできます。
4. **［保存する］** → **［PDF 出力］** で印刷できます。

### 11.2 ポイント

- 1 人のお客さまに **複数の施工** をまとめた合算請求もつくれます。
- 「未入金 / 入金済」のステータスを切りかえると、あとで集計しやすくなります。
- お会計が現金以外（カード）の場合、POS（ポス）会計の機能でカード決済もできます（別途設定が必要）。

## 12. こんなときどうする？（よくある質問）

### Q1. 画面がフリーズして動かない
**A.** ブラウザの右上にある「再読み込みマーク（くるくるの矢印）」を押してください。それでもダメなら、ブラウザを一度閉じて開きなおします。**入力中のデータは消えてしまう** ことがあるので、長く入力するときは途中で **［保存する］** を押す習慣をつけましょう。

### Q2. 間違って違うお客さまの情報を消してしまった
**A.** 落ち着いてください。多くの場合、削除直後はもとに戻せます。お店の代表者にすぐ連絡してください。30 日以内ならサポートからの復元も可能です。

### Q3. 写真が大きくてアップロードできない
**A.** 1 枚 10 MB を超えると上げられません。スマホの設定で写真の **解像度** を「高」→「中」に下げる、またはサイズを小さくするアプリで縮小してください。

### Q4. お客さまから「証明書がメールで届かない」と言われた
**A.** 次の順番で確認します。
1. 顧客情報のメールアドレスがまちがっていないか
2. お客さまの「迷惑メール」フォルダに入っていないか
3. それでも届かないときは、**QR コード** か **PDF 印刷** でお渡しする

### Q5. パスワードを忘れた
**A.** ログイン画面の **「パスワードを忘れた方はこちら」** を押すと、登録したメールアドレスにリセット用のメールが届きます。

### Q6. 他のスタッフに使ってもらいたい
**A.** お店の代表者だけが「メンバー追加」ができます。**［設定］→［メンバー］→［招待］** からメールアドレスを入れて招待してください。

### Q7. 「権限がありません」と表示された
**A.** あなたのアカウントには、その操作の権限がありません。お店の代表者に「○○ができるようにしてほしい」と伝えてください。

### Q8. 古い iPad / 古い Windows PC で使えますか？
**A.** 5 年以上前の機種だと動かないことがあります。Chrome / Safari / Edge を **最新バージョン** にアップデートしてからお試しください。

## 13. 使いそうな言葉の意味

| 言葉 | 意味（やさしく） |
|---|---|
| **ログイン** | 自分のメールアドレスとパスワードを入れて、画面に入ること |
| **ログアウト** | 画面から出ること（ほかの人に見られないようにするため、終わったら必ず押す） |
| **ダッシュボード** | お店全体のホーム画面 |
| **証明書** | 「いつ・誰に・どんな施工をしたか」を残すデジタルの紙 |
| **PDF（ピーディーエフ）** | 印刷もメール送信もできる便利なファイルの形 |
| **QR コード** | 四角いモザイク模様。スマホで読みこむとページがすぐ開く |
| **アップロード** | 写真やファイルを Ledra に登録すること |
| **ダウンロード** | Ledra から自分のパソコンに落とすこと |
| **ブラウザ** | Chrome や Safari など、インターネットを見るためのソフト |
| **アカウント** | あなた専用の入口（メールアドレスとパスワードのセット） |
| **テナント** | あなたのお店のデータの「いれもの」。他のお店からは見えません |
| **メンバー** | お店の中で Ledra を使うスタッフのこと |
| **API（エーピーアイ）** | お店の人は気にしなくて OK。他のシステムとつなぐための入口 |

---

## 困ったら

- まずは **左メニュー → ダッシュボード** に戻ると安心です。
- それでも分からないときは、お店の代表者か、Ledra のサポート窓口にお問い合わせください。
- **無理に押しまくらないこと。落ち着いて、ひとつずつ。** 大事なデータは自動で保存されているので、画面を閉じても消えることはほとんどありません。

## このマニュアルの更新について

このマニュアルは、画面の表記や機能の追加にあわせて少しずつ更新しています。

- 「ここがわかりにくかった」「このボタンの説明がほしい」というご要望は、お店の代表者まで遠慮なくお伝えください。
- 最新版は社内の共有フォルダ、または `docs/admin-beginner-guide.md` にあります。
